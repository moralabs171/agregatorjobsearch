# Decisions Log

## 2026-06-20 — Источник данных: API Arbeitsagentur вместо скрейпинга

**Решение:** использовать публичное REST API Bundesagentur für Arbeit
(`rest.arbeitsagentur.de`, `X-API-Key: jobboerse-jobsuche`) как единственный
источник вакансий и Ausbildung.

**Почему:** один endpoint покрывает и вакансии (`angebotsart=1`), и Ausbildung
(`angebotsart=4`). Альтернативы (StepStone, Indeed, LinkedIn) запрещают
скрейпинг в ToS, имеют анти-бот защиту и юридический риск.

**Последствия для безопасности:** все исходящие запросы по HTTPS; нет хранения
учётных данных третьих сторон; API ключ публичный (не секрет).

## 2026-06-20 — Приватность бота через allowlist chat_id

**Решение:** бот обрабатывает команды только от `chat_id` из `ALLOWED_CHAT_IDS`;
запуск без заполненного allowlist запрещён (ошибка конфигурации).

**Почему:** Telegram-бот публично доступен по имени; без фильтра им сможет
пользоваться кто угодно (нежелательная нагрузка на API, утечка интересов).

**Последствия для безопасности:** ограничение доступа на уровне приложения;
снижает риск злоупотреблений и работает как простейший rate-limit (один юзер).

## 2026-06-20 — Дедупликация и хранилище

**Решение:** SQLite, отдельная таблица `seen(subscription_id, refnr)`; при
создании подписки текущие результаты помечаются как viewed, далее шлётся только
новое.

**Почему:** избежать спама старыми объявлениями; `refnr` — стабильный
идентификатор из API. SQLite достаточно для одного пользователя.

**Последствия для безопасности:** все SQL-запросы параметризованы (защита от
SQL-инъекций); пользовательский ввод не конкатенируется в SQL.

## 2026-06-20 — Покрытие профессий: словарь + авто-перевод RU→DE

**Решение:** свободный ввод сначала ищется в словаре синонимов (рус/нем), при
отсутствии — для кириллицы делается авто-перевод на немецкий через бесплатный
API MyMemory (без ключа, через httpx). При 0 результатов бот даёт подсказки.

**Почему:** API Arbeitsagentur немецкоязычное; русские запросы вне словаря не
находились. Альтернативы: только большой словарь (конечный) или DeepL (нужен
ключ). MyMemory покрывает любые профессии без регистрации.

**Последствия для безопасности:** длина запроса ограничена (100 симв.); ответ
переводчика валидируется (тип, статус, фильтр мусора); ошибки/таймауты
перехватываются и не пробрасываются пользователю; вызов по HTTPS; ключей и PII
в запросе нет (служебный e-mail-плейсхолдер). Внешний сервис — не критичный путь:
при сбое используется исходный текст.
